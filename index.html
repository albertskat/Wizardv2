Instructions: Create a new file in your html editor - Brackets, Atom, etc. Save it and name it with a ".html" ending. Ex) TunaversePOA.html

Step 1: Choose your number of cards

Instructions: After copying the text, paste into your file. On each card:
* change the title in TWO locations
* add or delete the description
* replace "fa-file" with the Font Awesome icon you want for the front of each card (click for cheatsheet)
